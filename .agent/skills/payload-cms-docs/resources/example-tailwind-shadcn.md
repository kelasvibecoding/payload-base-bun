# Example: Tailwind CSS & Shadcn UI

Integrates modern utility-first CSS and component libraries into the Admin Panel.

## Key Features
- **Admin Styling**: Using Tailwind classes within custom components.
- **Component Libraries**: Using Shadcn UI for complex widgets (tabs, dialogs, etc.).

## Setup Pattern
1. **Importing CSS**: Loading Tailwind's `@tailwind base`, etc., into a custom layout/provider.
2. **Configuring Tailwind**: Ensuring the `tailwind.config.js` includes the Payload source files.

## Use Cases
- Creating a proprietary design system for the CMS.
- Polishing custom views to match the project's brand identity.
- Building complex interactive fields that require high-fidelity UI.
