# Accessibility

Payload is committed to ensuring that our admin panel is accessible to all users, including those with disabilities. We follow best practices and guidelines to create an inclusive experience.

We are actively working towards full compliance with **WCAG 2.2 AA** standards.

## Compliance Standards

| Standard | Status | Description |
| :--- | :--- | :--- |
| **WCAG 2.2 AA** | In Progress | Web Content Accessibility Guidelines (WCAG) 2.2 AA is a widely recognized standard for web accessibility. |

## Our Approach

- **Integrated Axe**: We use Axe within our e2e test suites to ensure long-term compliance.
- **Custom Utilities**: We use custom utilities to test keyboard navigation, window overflow, and focus indicators across components.
- **Manual Testing**: We perform manual testing with screen readers and other assistive technologies.

If you encounter any accessibility issues, please report them in the [GitHub Discussion](https://github.com/payloadcms/payload/discussions/14489) page.
