# Relume UI Patterns

This skill provides a reference for premium UI patterns. When building new components or layouts, refer to the files in the `library/` directory to ensure high-end aesthetics and modern UX.

## Core Principles
1. **Responsive First**: All components are designed to work seamlessly from mobile to desktop.
2. **Modern Interactions**: Use Framer Motion for smooth transitions and micro-interactions.
3. **Clean Architecture**: Components are structured for reuse and easy customization.

## How to use
- Browse the `library/` folder for specific categories (e.g., `shells`, `heroes`).
- Adapt the patterns to the project's specific tech stack (Next.js, Tailwind, Shadcn UI).
- Ensure consistency with existing design tokens in `index.css`.
